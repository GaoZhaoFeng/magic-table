# 在线表格系统前端
> 专注于高定制化在线表格管理~基于SpringBoot2.4 + Vue3 + Ant Design Vue2 + Ag-Grid + MongoDB
## 开发环境
> npm务必不要用7的版本，否则npm install的时候会直接报错
+ node:10.15.0
+ npm:6.14.11

## 相关命令
+ 安装依赖：`npm install`
+ 启动项目：`npm run serve`
+ 源码编译：`npm run build`
